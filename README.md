# DOOGY DAYS
> A pet care app that allows a user to track information about their pet(s)

## MVP

### `Sally`
Sally (my dog) is the default animal

### `Information`

_time_:
The time the record was created

_walk_:
Did she go for a walk

_pee_:
Did she pee on the walk

_poop_:
Did she poop on the walk

_food_:
What food did she get, if any

_initials_:
Who added the information

_notes_:
Any special notes about the entry
